# Foobar

Foobar is a Python library for dealing with word pluralization.

## Installation

```bash
git clone https://github.com/wrmn/goText
cd goText
go build goText.go
```

## How To Use
```bash
./goText <file-name> <command>
```
# Command
```bash
new
```
create new file
```bash
edit
```
Write or edit on file 
```bash
read
``` 
view file
```bash
delete
``` 
delete file
